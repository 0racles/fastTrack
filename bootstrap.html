Jumbotron

The jumbotron is a big box for calling special attention to some special content or information. The default colour of a jumbotron is grey and its corners are rounded. It also enlarges the font of the text inside it. Placing the Jumbotron outside the container makes it extend to the edges of the page

How Carousels work (1st step)

First you wrap the whole Carousel in a div. Give it an 'ID' a 'class' and a data-ride attribute. The data-ride attr will trigger the animation effect class (eg .slide) of the carousel immediately the page loads

How Carousels Work (2nd step)

create a set of indicators to show the positon of the active image. Use an 'OL'and give it a class .carousel-indicator.In the LI tags add a data-target atrr to point to the ID of the carousel. Then also include a data-slide-to attr setting them to each number of image

Hoe Carousels Work (3rd Step)

Then create a div with the class .carousel-inner and then nest all the immages or text in (sub) divs all with a .item class. If you want to label them simply add the .carousel-caption class to a new div just as we are doing now

How Carousels Work (4th Step)

Now create arrows to slide the carousel. in an 'a' for the left arrow link it to the carousel ID, then give it a data-slide attr with the value 'prev'. Do not forget to add a class of left carousel-sslide to it. Then put a span tag with a class of .glyphicon glyphicon-chevron-left and you can also include a span for screen readers. Repeat the same for the right arrow onnly changing the text from left to right where necessary
